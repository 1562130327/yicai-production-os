## 一、订单录入表单

### 1.1 通用区（必填/选填）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 客户 | 下拉搜索 + 手输 | ✅ | 选历史客户，新客户手输自动保存到客户表 |
| 款号 | 手输 | 否 | 产品编号 |
| 工艺类别 | 24种卡片按钮 | ✅ | 选中后右侧实时预览工序链 |
| 优先级 | 下拉 | 否 | 默认"客户没提"，5级颜色区分 |
| 用料 | 动态下拉 | ✅ | 格式 `供应商-材质-颜色-宽度-厚度（库存X张）`，只显示库存有的 |
| 产品尺寸 | 自动计算 | 否 | 规则：只有片材→片材尺寸，有切片无冲型→切片尺寸，有冲型→冲型后尺寸 |

### 1.2 尺寸区（按工艺联动）

| 工艺包含 | 显示字段 |
|----------|---------|
| 有片材 | 片材尺寸（宽×长×厚，三段式输入）、片材总数 |
| 有切片 | 尺寸行 × N 条（切片尺寸、切片数量），可动态增删 |
| 有冲型 | 尺寸行 × N 条（冲型后尺寸、冲型数量），可动态增删 |
| 有多层 | 尺寸行叠加，每层独立长宽厚 |
| 库存类 | 不显示片材字段 |

### 1.3 校验规则

- 冲型后尺寸 ≤ 切片尺寸
- 切片尺寸 ≤ 片材尺寸
- 必填项按工艺动态检查
- 尺寸自动标准化（任何分隔符→统一 × ）
- 数量 > 0
- 数量合理性：切片总数 ÷ 每张片材可切数超出理论范围 → 黄色警告
- 库存实时提醒：够用✓ / 不够⚠️

### 1.4 便捷功能

- 连续录单客户信息保留
- 订单号自动生成（ORD-年月日-序号）
- 复制已有订单并修改
- 提交前预览工序链
- 提交前确认卡

## 二、库存入库表单

### 2.1 必填区

| 字段 | 类型 | 规范 |
|------|------|------|
| 供应商 | 下拉选择 + 手输 | 历史供应商可选，新供应商自动保存 |
| 材质 | 下拉 + 手输 | 新材质自动保存 |
| 颜色 | 下拉 + 手输 | 新颜色自动保存 |
| 规格 | 三段式输入 | 宽(m)、长(m)、厚(mm) 三个数字框，自动拼 `Xm×Ym×Zmm` |
| 入库数量 | 数字 | 单位"床"，>0 |

### 2.2 选填区

| 字段 | 说明 |
|------|------|
| 单价 | 元/床，可选 |
| 总金额 | 自动=单价×数量，也可手输 |
| 备注 | 批次号、采购单号 |

### 2.3 规范保障

- 三段输入杜绝格式不统一
- 供应商/材质/颜色联动
- 新材料保存后下次出现在下拉选项
- 提交前确认卡

## 三、报表系统

### 3.1 库存报表

| 报表 | 内容 | 权限 |
|------|------|------|
| 日报 | 今日入库/出库明细 + 当前库存快照 | 管理员+跟单员 |
| 周报 | 周入库/出库按供应商汇总 + 预警清单 | 管理员+跟单员 |
| 月报 | 月度进销存（期初→入库→出库→期末）+ 成本合计 | 管理员 |
| 年报 | 年度材料消耗趋势 | 管理员 |

### 3.2 订单报表

| 报表 | 内容 | 权限 |
|------|------|------|
| 月报 | 当月新增/完成/取消订单统计 + 按客户汇总 | 管理员+跟单员 |
| 年报 | 年度订单量趋势 + 客户排名 | 管理员 |

## 四、数据模型变更

### 4.1 订单新增字段

| 字段 | 类型 | 说明 |
|------|------|------|
| dimensions | JSON | 尺寸行数组 `[{sliceSize, sliceQty, punchSize, punchQty}]` |

### 4.2 库存新增字段

| 字段 | 类型 | 说明 |
|------|------|------|
| color | TEXT | 颜色 |
| width | REAL | 宽度(m) |
| price | REAL | 单价(元/床)，可选 |

### 4.3 报表所需

库存出入库流水表已存在 (`inventory_transactions`)，日报/周报/月报通过聚合查询实现。

## 五、前端页面

### 5.1 新增/修改页面

- 订单录入页（基于工艺联动切换字段）
- 库存入库页（三段式规格输入）
- 材料统筹页（多订单联合开片方案）
- 报表页（日报/周报/月报/年报切换）

### 5.2 联合开片（材料统筹）

跟单员录单后，系统记录剩余料头。管理员在材料统筹页：
1. 选择需要统筹的订单（同供应商+同材质+同颜色优先同宽度）
2. 系统列出联合开片方案
3. 打印方案给相关师傅
4. 方案选定后锁定用料分配
